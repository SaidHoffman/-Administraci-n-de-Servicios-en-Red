# Administración de Servicios en Red

Este repositorio contiene el proyecto de laboratorio de GNS3 en la carpeta `Proyecto/`.

## Estructura

- `Proyecto/` — archivo del proyecto GNS3 y archivos asociados
- `Proyecto/README.md` — información adicional del laboratorio

## Nota

La topología requiere una imagen de IOS válida para poder ejecutarse en GNS3.
